AZ-900 Flashcards (2025)

Key Azure Fundamentals & Scenario Practice

What are the three main cloud service models?
IaaS, PaaS, SaaS
What are the three deployment models?
Public, Private, Hybrid cloud
Scenario: A company wants to manage hardware themselves but use Azure networking & storage. Which model?
IaaS (Infrastructure as a Service)
Scenario: Developers want to build apps without managing servers. Which model?
PaaS (Platform as a Service)
Scenario: End users need access to Microsoft 365 apps online. Which model?
SaaS (Software as a Service)
Which service provides scalable virtual servers?
Azure Virtual Machines
Which service offers serverless compute?
Azure Functions
Which service provides managed Kubernetes?
Azure Kubernetes Service (AKS)
Which service stores unstructured data (blobs)?
Azure Blob Storage
Which service is for globally distributed NoSQL databases?
Azure Cosmos DB
Scenario: Store user profile images for a social app. Which storage option?
Azure Blob Storage
Scenario: Global e-commerce app needs fast data replication worldwide. Which DB?
Azure Cosmos DB
What does Azure AD provide?
Identity & Access Management, SSO, MFA
Which tool provides role-based access control (RBAC)?
Azure RBAC
Scenario: A company requires MFA for all employees. Which service enforces this?
Azure Active Directory
Scenario: Limit a user to read-only access in storage. Which feature?
RBAC (Reader role)
Which service lets you organize resources into groups?
Resource Groups
Which service helps enforce policies (e.g., only deploy in certain regions)?
Azure Policy
Which service provides a dashboard & unified management view?
Azure Portal
Scenario: Prevent developers from deploying VMs in non-compliant regions. Which tool?
Azure Policy
Scenario: Track performance & alerts across all resources. Which service?
Azure Monitor
What tool helps estimate Azure costs before deployment?
Azure Pricing Calculator
What tool helps track actual usage and spending?
Azure Cost Management
Scenario: Finance team wants to forecast monthly Azure spend. Which tool?
Pricing Calculator
Scenario: CFO needs detailed cost analysis by resource group. Which tool?
Cost Management + Billing
What does SLA define?
Guaranteed uptime/availability (e.g., 99.9%)
Scenario: Service with 99.95% SLA. Approx downtime allowed per month?
~22 minutes/month
Which Azure tool provides compliance reports & certifications?
Microsoft Trust Center / Compliance Manager
Which service is used for disaster recovery & geo-redundancy?
Azure Site Recovery
Scenario: Company wants backup copies across regions for resiliency. Which option?
Geo-redundant storage (GRS)
